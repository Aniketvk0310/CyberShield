# Development Phase

## Technologies
- HTML5, CSS3, JavaScript
- Bootstrap 5 for layout and responsiveness
- Three.js for 3D model rendering
- AOS for scroll animations
- Font Awesome for iconography

## Asset Production
- Bitmap images: designed and exported as PNG with cyber-themed visuals.
- Vector images: custom SVG logo and icons created for reuse.
- Audio: alert clip edited for clarity and consistent volume.
- 3D model: procedural shield built directly in Three.js.

## Implementation Steps
1. Created the folder structure and base layout.
2. Built common UI components (navbar, footer, hero sections).
3. Added interactive modules (quiz, password checker, phishing detector).
4. Integrated the 3D lab with Three.js and OrbitControls.
5. Added audio awareness section with controls.
6. Applied animations, glassmorphism, and responsive styles.

## Integration Notes
- All multimedia assets are referenced locally from images/ and audio/.
- 3D lab rendering is confined to the container for reliable layout behavior.
- JavaScript is split into focused modules for easier testing and maintenance.
